# Lightning FastAPI

<p align="center">
    <em>Lightning FastAPI framework, provides boiler plates for FastAPI based on Django Framework</em>
</p>

## Explaination

```
/
|
│   manage.py
│   README.md
│   requirements.txt
│
└─── api
│   │   helpers.py
│   │   settings.py
│   │   urls.py
│
└─── accounts
    │   urls.py
    │   views.py
```

## Installation

<div class="termy">

```console
$ pip install -r requirements.txt

---> 100%
```

</div>

Run the server

<div class="termy">

```console
$ python manage.py
```

</div>

Open http://127.0.0.1:8000/docs
